○LOGOの文法について

こちらの Blogエントリーにまとめたのでご参照ください

  Logo基礎文法最速マスター :接触編
  http://d.hatena.ne.jp/minekoa/20100203/1265202628

LOGO は Lisp の一種としての言語コアに
普通の言語風の構文の見た目を実現する糖衣層をかぶせた形をしているので、
普通の言語のようにみえるプログラミング言語の中では
比較的容易に実装できます。

LOGO の言語コア層のシンタックスは以下で全てです

>||
LOGOプログラム ::= { コマンド }*
コマンド       ::= 手続き名 + { 入力 }*
入力           ::= "語 | リスト | オペレーション
オペレーション ::= 手続き名 + { 入力 }*
語             ::= { 英数字 | 特殊記号 }*
リスト         ::= { { 語 | リスト }* }
||<

インタプリタ実装初心者が読みながら楽しんで実装できるよう
配慮したつもりです。

それでは、言語実装をお楽しみください。
